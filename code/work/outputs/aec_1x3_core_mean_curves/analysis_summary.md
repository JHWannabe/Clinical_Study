## 1. 스크립트 개요

- **연구 질문** (스크립트 자체 docstring): 여러 날에 걸쳐 내부 데이터(g1090)에서 찾은 4개 해부학적 구간(R1~R4) AEC 형태 게이트를, 한 번도 보지 않은 외부 데이터(sdata)에 그대로 적용했을 때 최종 그림으로 봐도 이야기가 되는가?
- **1단계 게이트**: 임상변수만으로 만든 점수를 내부 데이터의 90% 민감도 지점(S90)에서 임계값으로 고정 → 외부 데이터에서 "임상 양성/음성" 분리.
- **2단계 게이트**: R1(45–56)/R2(57–80)/R3(97–128)/R4(117–128) 4개 구간의 형태 서술자를 내부 데이터 통계로 z-표준화 후 외부에 적용, 임상 임계값 근처에서만 작동하는 가우시안 가중 투표로 `"+/-"` 4자리 패턴을 만들고, 사전 선택된 5개 패턴(`++++, ++--, +--+, --+-, ---+`)에 해당하면 "AEC+"(하향조정 후보)로 분류.
- **출력**: 1x3 평균곡선(A/B/C), R4 기울기 주석 버전, 2x3 평균+미러편차 버전, 요약 CSV 2개, JSON 1개.

## 4. ⚠️ 범례에 가려져 있던 데이터 (재확인·재계산 결과)

이미지를 패널별로 잘라 확대한 결과, **2x3 그림의 하단 E 패널(실제 저SMI vs 비저SMI, 전체평균 대비 편차)** 에서 범례(`loc="upper left"`)가 **곡선이 가장 높은 시작 구간(슬라이스 1~30)** 과 정확히 겹칩니다. 이 구간은 CSV 요약(`external_s90_core_2x3_mirror_deviation_summary.csv`)에도 R1~R4 구간만 집계돼 있어 수치가 아예 빠져 있었습니다. 원본 데이터(`g1090.xlsx`/`sdata.xlsx`)를 다시 로드해 동일 함수로 재계산한 결과:

**중요한 발견**: E패널에서 Low SMI+ 그룹의 편차는 **슬라이스 1번(치골결합 하연 바로 위)에서 이미 0.0353**으로, 전체 그림(모든 패널·모든 슬라이스) 최댓값인 0.0380(R3, 109번 슬라이스)에 거의 맞먹는 수준입니다. 즉:

- 실제 저SMI 표현형은 **R1(45–56)이 시작되기도 전인 1~44번 구간**(골반 하부~치골결합 인접부)에서부터 이미 강한 분리 신호(평균편차 0.0182)를 보이는데, **현재 게이트의 4개 브랜치(R1~R4) 중 이 구간을 사용하는 것은 하나도 없습니다.**
- 같은 구간에서 임상+/− 대비(D패널)는 분리가 훨씬 약함(평균 0.0057~0.0078) → **임상 점수는 이 초반 구간의 신호를 거의 못 잡아내는데, 실제 표현형은 여기서부터 갈라진다**는 뜻. 형태 게이트가 놓치고 있는 잠재적 추가 정보 영역.
- 참고로 R2와 R3 사이(81~96번, 현재 게이트가 쓰지 않는 또 다른 공백 구간)도 재계산해보면 Low SMI+ 평균편차 0.0135, 최대 0.0295(96번)로 역시 무시할 수 없는 신호가 있음(F패널의 AEC+/- 대비는 이 구간에서는 미미: 평균 0.0063/0.0007).

**결론**: 범례가 가린 것은 단순한 "미관상 문제"가 아니라, **1~44번 구간(그리고 부차적으로 81~96번 구간)에 R1~R4 게이트가 놓친 실제-표현형 분리 신호가 존재한다는 것을 시각적으로 확인하기 어렵게 만든 것**입니다. 이는 아래 5절의 "추가 발전 방안"에 새로운 후보 영역(슬라이스 1–44)으로 반영할 가치가 있습니다.

## 5. 종합 판단

1. 1단계 임상 게이트(S90)는 외부에서도 설계 목표(민감도 ~93%)를 재현 — 전이 잘 됨.
2. 2단계 AEC 형태 게이트는 임상 양성군의 10.4%를 사건율 26.9%→3.6%로 유의하게 분리(p<0.001), 민감도 손실은 1.4%뿐 — de-escalation 후보 발굴이라는 목표가 외부 검증에서도 성립. 같은 패턴이 내부(g1090, 22.2%→3.8%, p=5.5×10⁻⁴)에서도 방향·크기 모두 재현되어(2-1절), 외부 결과가 우연이 아님을 뒷받침.
3. R3~R4(후반부) 형태, 특히 R4 끝점 기울기가 이 분리의 핵심 동력이며, 이는 실제 저SMI 표현형의 "R3 융기" 신호와 방향이 일치 — 우연이 아닌 형태학적 신호일 가능성.
4. **(신규)** 슬라이스 1~44 구간에 현재 게이트가 활용하지 않는 추가 분리 신호가 존재 — R1~R4 게이트가 잡아내지 못하는 잔여 정보가 있을 수 있음.

## 6. 추가 발전 방안 (우선순위순)

1. **AEC+ 서브그룹(n=56, 사건 2건) 불확실성 정량화** — 환자 단위 부트스트랩으로 3.6%/26.9% 및 Fisher p의 CI·안정성 확인.
2. **SELECTED_PATTERNS 선정 과정의 과적합 점검** — 내부 데이터 split-half/재표collection으로 5패턴이 재현되는지 검증.
3. **R3/R4 중심의 규칙 단순화** — 분리 신호가 R3~R4에 집중된 만큼, 축소된 1~2 브랜치 규칙으로도 동일 효과가 나는지 비교.
4. **성별·사이트/벤더 층화 재검증** — 과거 site/vendor 감사 파일들의 우려를 이번 최종 게이트에도 재적용.
5. **통계적 유의성 → 임상적 유용성 전환** — NNT/NNH, decision curve analysis로 실제 활용 가치 제시.
6. **(신규) 후보 구간(슬라이스 1–44) 및 81–96 공백 구간 탐색** — 본 재검토에서 드러난, 현재 게이트가 쓰지 않는 구간의 실제-표현형 분리 신호를 형태 서술자로 추가해 게이트에 통합할 가치가 있는지 확인.
